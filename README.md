## Running the Application
### Client
#### Local
- In terminal, navigate to the `client` folder
- With Python installed, run:
```
python -m SimpleHTTPServer 8000
```
- Local application is now served at:
```
http://localhost:8000/
```

## Running the SPRING
```java
mvn jetty:run
```
* URL : `http://localhost:8080/spring3`
